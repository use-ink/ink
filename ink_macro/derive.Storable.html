Derive Macro Storable

Source 
#[derive(Storable)]
Expand description

Derives ink::storage’s Storable trait for the given struct, enum or union.

§Examples

use ink::storage::traits::Storable;

#[derive(Storable)]
struct NamedFields {
    a: u32,
    b: [u32; 1],
}

let value = <NamedFields as Storable>::decode(&mut &[123, 123][..]);
