Macro ink_lang_ir::format_err[−][src]

macro_rules! format_err {
    ($spanned:expr, $($msg:tt)*) => { ... };
}

Creates a syn::Error with the format message and infers the Span using Spanned.

Parameters

  • The first argument must be a type that implements syn::spanned::Spanned.
  • The second argument is a format string.
  • The rest are format string arguments.

Note

On stable Rust this might yield worse error span information to the user than format_err_spanned.
