The contract definition.

Dispatch

Constructor Dispatch

  1. Retrieve function selector s.
  2. Find selected constructor C given s:
    • If C was found:
      1. Decode input arguments i according to C's requirements
      2. Initialize dynamic storage allocator for instantiation
      3. Call C(i), returns storage instance S
      4. Push state of S to contract storage
      5. Finalize dynamic storage allocator
      6. Exit
    • Otherwise:
      1. Panic that the provided constructor selector was invalid

Message Dispatch

  1. Retrieve function selector s.
  2. Find selected message M given s:
    • If M was found:
      1. Decode input arguments i according to M's requirements
      2. Initialize dynamic storage allocator for calls
      3. Pull storage instance S from contract storage
      4. Call M(s, i), returns return value R
      5. If M has mutable access to the contract storage:
        1. push mutated state of S back to contract storage
      6. Finalize dynamic storage allocator
      7. If R is not of type ():
        1. return R
      8. Exit
    • Otherwise:
      1. Panic that the provided message selector was invalid
