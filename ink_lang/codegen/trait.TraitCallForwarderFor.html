Expand description

Implemented by call builders of smart contracts.

These might be implementing multiple different ink! traits. The codegen makes them implement this trait once for every ink! trait they have to implement.

While the trait is not necessary it encapsulates a lot of utility and auxiliary code required for the actual ink! trait implementations.
