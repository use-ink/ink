pub trait Allocate {
    fn alloc(&mut self, size: u64) -> Key;
}

Types implementing this trait can allocate storage.

Note

Since the current Wasm implementation is 32-bit we are fine to only support allocation sizes of max 32-bit in contract storage. However, for static allocator like BumpAllocator that is meant to allocate also other allocators we might need relaxed allocation sizes.
